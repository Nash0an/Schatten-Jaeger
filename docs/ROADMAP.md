# Roadmap: Schatten-Jäger

Dieses Dokument hält die nächsten sinnvollen Ausbauschritte fest. Es beschreibt keinen garantierten Plan, sondern die aktuelle Priorisierung.

## Prio 1: Stabilität und Produktisierung
- [ ] GitHub-Release vorbereiten: README, Lizenz, Screenshots und saubere Commit-Historie prüfen.
- [ ] Manuelle Test-Checkliste für Desktop und Touch als Markdown ergänzen.
- [ ] Menü- und Progressionspfade nach weiteren Edge Cases prüfen.

## Prio 2: Gameplay und Feedback
- [ ] Weitere Soundvarianten für Win/Lose/Spawn ergänzen.
- [ ] Neue Gegnertypen einführen: schnell, fragil oder träge, aber groß.
- [ ] Power-Ups testen, z. B. längerer Schatten oder kurzzeitige Schockwelle.

## Prio 3: Tools und Inhalte
- [ ] Einfachen Level-Editor oder JSON-Import-Konzept skizzieren.
- [ ] Balancing-Pass für späte Level dokumentieren.
- [ ] Optionalen Party-/Mehrspieler-Prototyp auf Basis des vorhandenen Koop-Modus untersuchen.

## Langfristig
- [ ] Online-Highscores.
- [ ] Gamepad-Unterstützung.
- [ ] Portierung auf Mobile-Wrapper wie Capacitor.
